# How to Use This Repo

Follow the docs in order.

Recommended path:
1. project context
2. learning objectives
3. repo structure
4. Unity Catalog setup
5. landing volumes and raw files
6. Auto Loader bronze ingestion
7. src modules and imports
8. silver transformations with `df.transform()`
9. UDFs and business rules
10. gold tables and views
11. permissions and governance
12. validation
13. Catalog Explorer walkthrough
14. Jobs, tasks, and optional schedule
15. final expected result
16. Lakeflow preview

Do not jump randomly. Each stage builds on the previous stage.
