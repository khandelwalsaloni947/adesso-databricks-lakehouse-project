# src Modules and Imports

This project stores reusable transformation logic in `src/`.

The notebooks should add the src path like this:

```python
import sys
sys.path.append("/Workspace/Repos/your-user/adesso-week8-day4-mini-project/src")
```

Then modules can be imported from:
- `transformations/`
- `udfs/`

This keeps notebook logic cleaner and more modular.
