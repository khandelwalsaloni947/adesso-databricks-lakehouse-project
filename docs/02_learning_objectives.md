# Learning Objectives

After finishing this project, you should be able to:
- explain the Unity Catalog hierarchy
- create catalogs, schemas, volumes, tables, and views
- distinguish governed file storage from governed table storage
- use Auto Loader to ingest raw data into bronze
- use `sys.path.append(...)` and import code from `src/`
- apply `df.transform(...)` for modular transformations
- use a UDF where custom business logic is needed
- create gold outputs
- apply basic privileges and inspect grants
- inspect governed objects in Catalog Explorer
